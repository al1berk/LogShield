# Presentation Demo

This folder contains demo inputs used by the Streamlit dashboard.

- `sample_logs.json`: curated benign, malicious, obfuscated, evasion, and hard-negative log lines.

Recommended demo path:

1. Start the dashboard with `make dashboard`.
2. Open `http://localhost:8501`.
3. Use the `Demo Console` view to scan curated examples.
4. Open `Model Results` to show regex-vs-BiLSTM evaluation artifacts.
5. Open `Dataset Quality` to show split and obfuscation coverage.
